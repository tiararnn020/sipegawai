# SiPegawai — Sistem Manajemen Data Pegawai

Aplikasi web manajemen data pegawai yang dibangun sebagai proyek uji kompetensi Junior Web Programmer LSP UPN Veteran Jawa Timur.

## Teknologi yang Digunakan

- PHP Native
- MySQL
- Bootstrap 5
- Chart.js
- PhpSpreadsheet

## Fitur

- CRUD Data Pegawai
- Dashboard dengan 3 Grafik Statistik
- Live Search Real-time
- Filter Kolom (Jenis Kelamin, Pendidikan, Usia)
- Export Data ke Excel (.xlsx)
- Print Data
- Modal Konfirmasi Hapus Dua Tahap
- Implementasi OOP (Class Pegawai)

## Cara Menjalankan

1. Clone repository ini
2. Letakkan folder di `C:\xampp\htdocs\`
3. Import database: buka phpMyAdmin, buat database `db_pegawai`, import file SQL
4. Install library: buka terminal di folder project, jalankan `composer install`
5. Aktifkan XAMPP (Apache + MySQL)
6. Buka browser: `http://localhost/pegawai`
